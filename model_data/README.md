# 模型文件数据

market1501.pb、mars-small128.pb 这两个是目标检测的特征文件（必须）

yolov4.weights 是网络的权重文件，由它生成yolov4.h5的。

```text
链接：https://pan.baidu.com/s/1qsi_rSqkWw0-pS3CPe-Mfg   提取码：11d0
```

下载好的文件，放到工程的model_data 目录中

```text
文件目录内
...
├── model_data 
│   ├── market1501.pb
│   ├── mars-small128.pb
│   ├── yolov4.h5
│   ├── yolov4.weights
│   └── README.md
...
```
